## Project Overview

hyxrestration — prediction-market strategy-testing lab (`hyxlab/`).
Python 3.14 + DuckDB. Archives Kalshi/Polymarket market data, replays it
through a no-lookahead simulator, and kills strategies via pre-registered
backtests. Phase 0 (falsified ag-equity thesis) is a closed historical
record under `phase0/` — do not build on it.

## Commands

```bash
# Test (73 tests; test-gate hook runs this on stop)
.venv/bin/python -m pytest tests/ -q

# Lint/Format (auto-format hook runs ruff on edit)
.venv/bin/ruff check hyxlab tests

# Archive health / one collection cycle / backtest replay
.venv/bin/python -m hyxlab.sweep --doctor
.venv/bin/python -m hyxlab.collect --once
.venv/bin/python -m hyxlab.run_backtest
```

Scheduled: systemd user timers `hyxlab-collect` (5 min) and
`hyxlab-sweep` (daily 06:10 UTC). DuckDB is single-writer — ad-hoc runs
may need to wait seconds for the collector's flock.
